# controle-de-gastos

# MVP
Por enquanto um modelo menor, mais enxuto, porem funcional.
## Objetivo - MVP
O usuario deve cadastrar suas receitas e gastos, podendo vizualizar estatisticas mensais e acompanhar seu saldo
## Usuario e Autenticação - MVP
- Cadastro com nome, email e senha 
- Login
- Logout
## Receitas - MVP
- Cadastro com valor, descrição e data
- Listar receitas (todas e por mes/ano)
- Editar / Excluir receitas
## Gastos - MVP 
- Cadastro com valor, descrição, data, categoria (alimentação, moradia, transporte, lazer, outros)
- Listar gastos (todas e por mes/ano)
- Editar / Excluir gastos
## Dashboard - MVP
- Total de receitas, total de gastos, saldo *(receitas - gastos)*
- Gastos por categoria (grafico de pizza)
